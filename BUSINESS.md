# Business: task-management
>
> ⚠️ IMMUTABLE BY AI.

## Goal

MCP todo server and web UI for unified task management across projects (see repo `README.md`).

## Constraints

- Do not store secrets in repo; use `env.example` and local `.env` only.
- AI must not delete or merge user tasks without explicit instruction.

## Consumers

- Developers using Cursor IDE and MCP clients.
